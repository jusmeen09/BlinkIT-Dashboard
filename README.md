# BlinkIT Dashboard

Data analysis and visualisation project for BlinkIT using the provided Excel dataset.

## Project structure
- `data/` — raw dataset: BlinkIT Dashboard.xlsx
- `notebooks/` — Jupyter notebooks for analysis
- `reports/` — final report and screenshots
- `README.md`, `.gitignore`, `LICENSE`

## How to run
1. Open `"C:\Users\hp\OneDrive\Desktop\BlinkIT Dashboard.xlsx"` in Excel or import it into Power BI.


## Goals
- Extract KPIs: total sales, orders, avg. order value
- Visualise category-wise revenue and outlet performance
- Build an interactive dashboard (Power BI / Excel)
